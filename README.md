# Sistema de Controle de Produtos (Python)

Projeto simples desenvolvido em Python com objetivo educacional.

O sistema funciona via terminal e permite:

- login de administrador
- criação de contas de usuário
- cadastro de produtos
- listagem de produtos
- navegação por menu interativo

## Objetivo do projeto

Este projeto foi criado como prática de:

- lógica de programação
- estruturas condicionais
- listas e dicionários
- funções
- tratamento de erros com try/except
- organização de código em Python

## Tecnologias utilizadas

Python 3

## Como executar

Clone o repositório:

git clone https://github.com/seu-usuario/controle-produtos-python.git

Entre na pasta:

cd controle-produtos-python

Execute:

python main.py
